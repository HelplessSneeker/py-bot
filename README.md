## About The Project

  This project was created for the associated boot.dev course. 

  It enables gemini to
  - read file content and infos
  - write to files
  - execute python files


### Prerequisites

  To execute the project the dependencies [uv](https://github.com/astral-sh/uv) Package and Project manager will be needed

### Usage

1. Clone the repo
   ```sh
   git clone https://github.com/HelplessSneeker/py_bot.git
   ```
2. then run the program file
  ```sh
   uv run main.py [prompt]
```
